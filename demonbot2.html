FOMO Machine

Demon Bot #2

The second piece in my series “Demon Bots”, “FOMO Machine” is a chrome extension that, whenever you try to log into social media, presents you with 9 images mined from your social media feeds and provokes you to select all the images that seem better (more fun, more exciting, more productive etc) than your day thus far in order to prove that you are indeed not a robot. Select any of the images and the captcha results will be inconclusive.



Select none and you can proceed to your desired destination. Alternatively, if you desire, you can outsource the expressing of your insecurity to a bot which will select images for you. In this manner, “FOMO Machine” explores whether the feelings of insecurity that social media encourages us to addictively dwell on may condition our online selves to become more predictable, impressionable or even programmed.

Fall 2021
    The installation consists of two parts: A chrome extension running in the user's browser and a python script running a local server on the user's computer.

    Set-up:
  • The local python server is responsible for generating the captcha images
  • The user enters their instagram login info as an input to the python script and, upon running the server, the script will query instagram for a list of all accounts the user follows.
  • The script then downloads the profile picture of each followed account, checks if there is a picture of a human in the profile picture using openCV and then deletes the downloaded image.
  • If a human is detected, then the queried account's instagram profile ID number is saved to a list of followed accounts in the python script's directory.
  • By confirming that a human is in the profile picture of each followed user, the python script ensures that it has a list of users whose posts will likely contain photographic content rather than illustration or infographic content which would likely be less FOMO inducing.
  • The python script user's login info remains on their computer locally and the list of their followers also is only saved locally as an anonymized list of numeric codes that only Instagram's API can interpret. Nothing can be accessed by the user that they cannot already access through instagram's website, and the user's data is not accessible by any remote source outside their computer.
  • The follower list only needs to be populated once - when the user first starts the FOMO machine.
    • Usage:
  • The chrome extension triggers whenever the user navigates to a page with an "instagram" or "facebook" domain.
  • The user is redirected to a custom, locally hosted web-page with the FOMO captcha on it, modelled using modified HTML and CSS from here.
  • The chrome extension sends an HTTP request to the python server, indicating that it should generate images for the captcha.
  • The chrome extension also sends the python script the URL that the user was attempting to access before being redirected. The python script saves this URL for when the user completes the captcha.
  • While the python script is generating the captcha images, the chrome extension's captcha webpage displays with a randomized FOMO-inducing prompt and 9 loading images in place of the captcha images.
  • The python script randomly picks three accounts from the user's followed account list and downloads each account's 5 most recent posted images using this instagram scraping code as a foundation.
  • The number of unique colors present in the 5 images are calculated and the first three images with more than 15,000 unique colors are selected, as these images are most likely to be photographs rather than graphics.
  • If all the downloaded images have less than 15,000 unique colors, then the first three images from the account's posts are selected.
  • Then the three selected images from each of the three accounts queried are selected and forwarded back to the chrome extension, where they are displayed for the user.
  • If the user clicks on the checkbox for any of the images and then clicks "I am not a robot", they will be redirected to another page indicating that their test results were inconclusive. The user is then redirected back to the captcha page.
  • If the user selects none of the images, or clicks on the "Help" button, which randomly selects from the 9 images, then the user will pass the captcha test and be redirected to a page indicating that they have been verified.
  • The verification page sends an HTTP request to the python script to receive the URL that the user was originally trying to access. The python script responds and the user is redirected to their desired destination.
  • Lastly, the python script then deletes all downloaded images from instagram as well as the stored URL the user was trying to access.
