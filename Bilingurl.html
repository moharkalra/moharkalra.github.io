Biling-URL

Background

Biling-URL was a chrome extension built at the Hackathon HackBCA III in 2016. Built with two other high school classmates, the chrome extension won Best Eucational Hack and Best Humanities Hack out of the 500+ attendees. We were unfortunately unable to continue work on Biling-URL after the hackathon, so the chrome extension remains unfinished.

Objective

Design a chrome extension that would help users learn a new language by immersing them in digital environment where that new language was used. As such, we wanted to build a chrome extension that would start with 10 or so words in the first week of use and translate all instances of those words into the language the user wants to learn on all pages that the user frequents online. Over weeks, the repository of words that would translated would expand, ultimately expanding to the translation of complete sentences and culminating with the translation of entire pages.

Design

    Parameters:

  • Javascript
  • Parse through webpage
  • Transmit select words through translation API
  • Replace those words on page with translations
  • Display in some manner words replaced and their translations.
  • Increment pool of words to translate with time
  • Ultimately translate entire sentences and phrases

    • Before and After Running the Extension:

    Solution:

  • Built chrome extension that parses through webpages
  • Finds all instances of 10 initially selected words for translation.
  • Passes words through Yandex API for translation and replaced words with translations on webpage
  • Displays all replaced words and translations in console for users’ reference
  • Began to implement functionality to increment pool of translated words over time
  • Time-progression functionality remains unfinished.
